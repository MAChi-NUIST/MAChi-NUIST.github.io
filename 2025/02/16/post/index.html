Markdown基础语法教程

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。本文将介绍Markdown的基本语法规则。

1. 标题

2. 文本格式化

2.1 强调

  • 斜体：使用一个星号或下划线包围文本 *斜体* 或 _斜体_
  • 粗体：使用两个星号或下划线包围文本 **粗体** 或 __粗体__
  • 粗斜体：使用三个星号或下划线包围文本 ***粗斜体***

2.2 删除线

这是删除线 ~~删除线~~

3. 列表

3.1 无序列表

使用 -、* 或 + 作为列表标记：

  • 第一项
  • 第二项
  • 第三项

3.2 有序列表

使用数字加点：

  1. 第一项
  2. 第二项
  3. 第三项

总结

以上就是Markdown的基础语法，掌握这些基本规则就能够编写出格式清晰的文档了。在实际使用中，建议参考这些规则，并多加练习。
